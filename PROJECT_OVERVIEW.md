# Обзор проекта: Aurora Optimizer

## 1. Концепция

**Aurora Optimizer** — это мобильное приложение для Android, разработанное как мощный инструмент для опытных пользователей, желающих оптимизировать производительность своего устройства. Ключевая особенность приложения — глубокая интеграция с **Android Debug Bridge (ADB)**, что позволяет выполнять системные операции, недоступные обычным приложениям.

Приложение помогает управлять установленными программами, удалять ненужное системное ПО (bloatware) и выполнять точечные настройки через ADB-команды.

## 2. Архитектура

Проект построен на современном технологическом стеке для Android и следует принципам чистой архитектуры.

- **Язык:** Kotlin
- **UI:** Jetpack Compose (декларативный UI-фреймворк)
- **Архитектурный паттерн:** MVVM (Model-View-ViewModel)
- **Навигация:** Jetpack Navigation for Compose
- **Внедрение зависимостей:** Hilt
- **Асинхронность:** Kotlin Coroutines и Flow

## 3. Подробная структура проекта

Проект имеет следующую иерархию пакетов, отражающую его архитектуру:

- `com.aurorabridge.optimizer`
  - `di`: Модули Hilt для внедрения зависимостей.
  - `model`
    - `AppItem.kt`: Модель данных, описывающая приложение.
    - `AppCategory.kt`: Enum для категорий приложений.
  - `repository`
    - `AppRepository.kt`: Репозиторий для получения списка приложений и информации о них.
    - `BackupHistoryRepository.kt`: Репозиторий для работы с историей бэкапов.
  - `services`
    - `BloatwareApps.kt`: Список известных bloatware-приложений.
  - `ui`
    - `AppNavigation.kt`: Логика навигации между экранами.
    - `theme`: Тема Jetpack Compose (цвета, типографика).
    - `vm` (ViewModel)
      - `AppManagerViewModel.kt`: Логика для экрана "Менеджер приложений".
      - `PermissionsViewModel.kt`: Логика для экрана "Разрешения".
      - `...` (другие ViewModel)
    - `screens`: Composable-функции для каждого экрана.
      - `AppManagerScreen.kt`: UI "Менеджера приложений".
      - `PermissionsScreen.kt`: UI экрана "Разрешения".
      - `HomeScreen.kt`: Главный экран.
      - `SettingsScreen.kt`: Экран настроек.
      - `...` (другие экраны)
  - `utils`: Вспомогательные классы и функции.

## 4. Основные возможности

### 4.1. Менеджер приложений

Это центральный компонент приложения. Он позволяет пользователям:
- **Просматривать все приложения:** С разделением на `Пользовательские`, `Системные` и `Bloatware`.
- **Фильтровать список:** Быстро находить нужные приложения по категориям.
- **Выполнять пакетные операции:** Выбирать несколько приложений для одновременного удаления или отключения.
- **Анализировать разрешения:** Просматривать полный список разрешений, которые запрашивает каждое приложение.
- **Управлять кэшем:** Очищать кэш для выбранных приложений.

### 4.2. Интеграция с ADB

- **Мастер настройки ADB:** Интерактивное пошаговое руководство, которое помогает пользователям активировать "Отладку по USB" и "Беспроводную отладку". Инструкции адаптируются под версию Android пользователя.
- **Консоль ADB:** Специальный экран, позволяющий напрямую выполнять ADB-команды прямо из приложения, что является уникальной функцией.

### 4.3. Дополнительные функции

- **Мастер оптимизации:** Пошаговый гид, который проводит пользователя через рекомендуемые шаги для улучшения производительности.
- **Логгер команд:** Экран для просмотра истории выполненных команд.
- **Локализация:** Приложение поддерживает русский и английский языки.

## 5. Уникальные "фишки"

- **Прямое выполнение ADB-команд:** Предоставление пользователю полного контроля над устройством без необходимости подключения к компьютеру.
- **Автоматическая классификация Bloatware:** Приложение использует внутренний список для идентификации нежелательного ПО от производителей.
- **Современный и адаптивный UI:** Использование Jetpack Compose обеспечивает плавный и интуитивно понятный интерфейс, включая поддержку темной темы.
